---
title: '钟薛高点评'
date: 2018-09-13
tags: ['诗', '2018']
---

**「钟薛高-酿红提」**

金谷风，落霞颂，层影两相重。

轻念余味及故踪，是这般殷红。

 <br/>
 
 <br/>

**「钟薛高-手煮茉莉」**

一曲烟雨雾朦胧，两分清词欲衔愁。

江畔千里香寒袖，霜庭万家酌古秋。

  <br/>
  
  <br/>

**「钟薛高-老树北抹茶」**

借君一言逍遥乐，

可得浮生乐逍遥。

  <br/>
  
  <br/>

**「钟薛高-丝绒可可」**

Life is transient. 

Sometimes ever, sometimes never. 

And you are the only soft echo.

  <br/>
  
  <br/>

**「钟薛高-爱尔兰陈年干酪」**

在那蔓布杜鹃、石楠、金雀与苔藓的岛屿，人们到处都在生活。

  <br/>
  
  <br/>

**「钟薛高-加纳黑金」**

臻醇之黑，是一抹无数个深夜唤起心弦，隐匿着一无所有与无所不有的金色光芒。

  <br/>
  
  <br/>

**「钟薛高-轻牛乳」**

你是耳边和煦的一语呢喃。

 <br/>
