---
title: '致歉'
date: 2018-07-15
tags: ['散文', '2018']
---

看到一张图，忘了从何而来，却想起相似的南邮操场。你们以为消失的瞬间，我总还记得，一首歌一种味道一个场景都能唤起。

<br/>


都说青春荒唐，只有在分道扬镳之时无畏聊出，才察觉到底有多荒唐。我以为时间能冲淡一切，正如它在我身上的人体现，其实不然。我曾经以为是保护自我、而故意去习惯的的尖酸刻薄，在别人眼中怕是刀子一般血淋淋。钉子敲了总会在铁板上留下伤痕，若非熔化重铸，怎能如新。

<br/>

那时的我觉得善意与爱慕如同四面八方的洪流，并非一支，太过沉重，害怕负担，不知所措，于是筑起冰墙，两耳不闻窗外事，一心只私观念诗。我现在完全可以理解之后外部的恶意从何而来，可我已经失去了道歉的资格与机会。

<br/>

没有见识过如此的我的人，怕是极难想象这样的我。再久远些的我一直弱气着，凭借着敏感的神经小心翼翼地维护每一层关系，若不是触及原则，断不会决绝。

<br/>

如今，我渐渐畏惧了毫无戒备的开朗，好似奉呈给大众的闯私宅的良机，于有意者总是不会错过的窥探。而在没有吃过恼的人们眼中，秘密成了我的个性，这也不失为一件乐观的阴差阳错了。

<br/>

倘若我真那么冰冷，也不会牢记这么多瞬间，重要的，不重要的，成为过去曲径康庄的一部分。当下我愈发觉着，坦诚才是所有问题的节点，人世已这般纷乱，不多些善意真挚，又怎能和平。

<br/>

为此，想向我冷语相对、恶眼相加的人们，郑重说声抱歉。每个阶段我都有一份我坚守的外在，也有一份我不改的内在心性。当然，我也时常感到欣慰。很多的你们已经成为她们的他们，而我还是我，像三年前球场上那次大家动人的胜利之后，听着《红色高跟鞋》，逆流隐入汹涌而来的人海。

<br/>
